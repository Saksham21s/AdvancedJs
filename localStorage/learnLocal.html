Local Storage is a web storage API that allows you to store key-value pairs in a web browser persistently. Data stored in local storage does not expire and remains available even after the page is refreshed or the browser is closed.

ðŸ›  Key Facts About Local Storage:

  • Stores data in key-value pairs.
  • Data persists even after page reloads and browser restarts.
  • Stored data is available only on the same origin (protocol + domain + port).
  • Storage limit is around 5MB per domain.
  • Data can be accessed using `localStorage.getItem("key")`.
  • To remove a specific key: `localStorage.removeItem("key")`.
  • To clear all stored data: `localStorage.clear()`.

  • Local Storage is not secure - It should not store sensitive data as it's accessible via JavaScript.
  • Local Storage is not automatically deleted - It persists until manually removed.
  • Local Storage is not suitable for large data - For complex data, use databases.
